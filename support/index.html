よくある質問

Q. .md ファイルが開けません

iOS の Files アプリで .md ファイルを長押し → 「開く」→ Mado を選んでください。または共有シートから Mado を選択することもできます。

Q. PDF として書き出したい

Markdown を開いた状態で、右上の「↗」ボタンをタップすると PDF として共有できます。

Q. テーマを切り替えたい

ホーム画面右上の「⚙」ボタンから設定を開き、「デザイン」セクションでお好きなテーマを選んでください。システム / ライト / ダーク / セピア / インク の 5 種類から選べます。

Q. 文字を大きくしたい

同じく設定画面の「デザイン」→「文字サイズ」で 4 段階に調整できます。iOS の Dynamic Type 設定にも対応しています。

Q. 共有エクステンションが使えるアプリは？

テキストを共有できる任意のアプリから利用可能です。ChatGPT、Claude、Safari でテキスト選択、Notes、Slack、Discord、メールなど。

Q. データはどこに保存されますか？

Mado は何も保存・送信しません。開いたファイルや貼り付けたテキストは画面表示のみで、外部にもデバイス上のストレージにも保存されません。

Q. サブスクリプションはありますか？

いいえ。Mado は買い切りです。一度ご購入いただければ、永続的にご利用いただけます。
